
# Prompt thiết kế giao diện Login / Register giống Coursera

**PROMPT:**

> Thiết kế giao diện **Authentication (Login & Sign Up)** cho một nền tảng học trực tuyến giống Coursera.
> Phong cách thiết kế **modern EdTech, tối giản, chuyên nghiệp, thân thiện với người học**.
>
> ---
>
> ## 1. Layout tổng thể
>
> * Layout dạng **centered authentication card**
> * Background sáng, tối giản
> * Bên trái có thể có **illustration học online**
> * Bên phải là **login / signup form**
> * Responsive cho desktop và mobile
> * Thiết kế clean giống các SaaS platform
>
> ---
>
> ## 2. Header
>
> * Logo platform ở góc trên
> * Có thể hiển thị text nhỏ:
>
>   * “Learn from top universities and companies”
>
> ---
>
> # Login Form
>
> **Tiêu đề**
>
> ```
> Log in to your account
> ```
>
> **Các trường nhập**
>
> * Email address
> * Password
>
> **Chức năng**
>
> * Checkbox: Remember me
> * Link: Forgot password
>
> **Nút chính**
>
> ```
> Log In
> ```
>
> **Social login**
>
> * Continue with Google
> * Continue with Facebook
> * Continue with Apple
>
> **Divider**
>
> ```
> OR
> ```
>
> **Link chuyển sang đăng ký**
>
> ```
> New to the platform? Sign up
> ```
>
> ---
>
> # Register / Sign Up Form
>
> **Tiêu đề**
>
> ```
> Join for free
> ```
>
> **Các trường nhập**
>
> * Full name
> * Email address
> * Password
>
> **Password rule**
>
> * At least 8 characters
> * Include letters and numbers
>
> **Nút chính**
>
> ```
> Join for Free
> ```
>
> **Social signup**
>
> * Sign up with Google
> * Sign up with Facebook
> * Sign up with Apple
>
> **Link chuyển sang login**
>
> ```
> Already have an account? Log in
> ```
>
> ---
>
> ## 3. UI Component Style
>
> * Rounded input fields
> * Large CTA button
> * Soft shadow card
> * Hover effect button
> * Icon trong input field
>
> ---
>
> ## 4. Color Palette
>
> * Primary: Blue (#2A73FF)
> * Background: White
> * Text: Dark Gray
> * Secondary: Light Gray
>
> ---
>
> ## 5. UX Enhancements
>
> * Real-time input validation
> * Password visibility toggle
> * Loading animation khi login
> * Error message rõ ràng
>
> ---
>
> ## 6. Illustration
>
> * Vector illustration về **online learning**
> * Student using laptop
> * Books, certificates, graduation icons
>
> ---
>
> ## 7. Responsive Behavior
>
> **Desktop**
>
> * 2 column layout
>
> **Mobile**
>
> * Form centered
> * Illustration thu nhỏ
>
> ---
>
> ## 8. Design Style
>
> * Modern SaaS
> * Minimal EdTech UI
> * Clean typography
> * Professional learning platform aesthetic
>
> ---
>
> **Output yêu cầu**
>
> * High fidelity UI
> * Login + Register screens
> * Clean grid layout
> * Modern education platform design
