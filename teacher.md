
# Prompt thiết kế giao diện Instructor Dashboard (Coursera Style)

**PROMPT:**

> Thiết kế giao diện **Instructor Dashboard cho một nền tảng học trực tuyến giống Coursera**, phong cách **modern EdTech SaaS dashboard, chuyên nghiệp, dễ quản lý khóa học và học viên**.

---

# 1. Layout tổng thể

* Layout dạng **Admin / Instructor dashboard**
* **Sidebar navigation bên trái**
* **Main content area bên phải**
* **Top navigation bar**
* Card layout rõ ràng
* Responsive desktop + tablet + mobile
* Background trắng hoặc xám nhạt

---

# 2. Top Navigation Bar

Thanh trên cùng gồm:

**Bên trái**

* Logo platform

**Ở giữa**

Search bar:

```
Search courses, students, analytics
```

**Bên phải**

Icons:

* Notifications 🔔
* Messages 💬
* Instructor avatar dropdown

Dropdown gồm:

* Profile
* Instructor settings
* Earnings
* Log out

---

# 3. Sidebar Navigation

Menu bên trái:

* 🏠 Dashboard
* 📚 My Courses
* ➕ Create Course
* 👩‍🎓 Students
* 📊 Analytics
* 💰 Earnings
* ⭐ Reviews
* ⚙ Settings

Menu active có **highlight màu xanh**.

---

# 4. Instructor Overview Section

Header:

```
Welcome back, Instructor 👋
Manage your courses and students
```

Thống kê nhanh dạng card:

Card gồm:

* Total Courses
* Total Students
* Course Rating
* Total Revenue

Layout:

```
[ Courses ] [ Students ] [ Rating ] [ Revenue ]
```

---

# 5. My Courses Management

Section:

```
Your Courses
```

Course card gồm:

* Course thumbnail
* Course title
* Number of students
* Rating ⭐

Buttons:

```
Edit Course
View Analytics
```

# 6. Student Activity

Section:

```

```
