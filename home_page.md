

# Prompt thiết kế giao diện trang chủ Coursera

**PROMPT:**

> Thiết kế giao diện **homepage cho một nền tảng học trực tuyến giống Coursera**, phong cách hiện đại, tối giản, chuyên nghiệp, responsive.
>
> **Layout tổng thể:**
>
> * Thiết kế landing page dạng education platform
> * Nền trắng, nhiều khoảng trắng (white space), màu chủ đạo xanh dương và trắng
> * Typography hiện đại, rõ ràng
> * UI tối giản giống các platform EdTech lớn
> * Grid layout rõ ràng, responsive desktop + tablet + mobile
>
> **1. Header / Navigation Bar**
>
> * Logo bên trái
> * Thanh search lớn ở giữa (Search for courses, skills, universities)
> * Menu: Explore, Online Degrees, Certificates, Careers
> * Nút Sign In và Join for Free ở bên phải
> * Sticky navigation
>
> **2. Hero Section**
>
> * Background sáng với hình minh họa học online
> * Headline lớn:
>   *“Learn without limits”*
> * Subheadline mô tả nền tảng học trực tuyến
> * CTA button:
>
>   * Join for Free
>   * Explore Courses
> * Thanh search khóa học ngay dưới headline
>
> **3. Trusted by Companies Section**
>
> * Logo các công ty lớn: Google, IBM, Meta, Amazon
> * Layout horizontal logo slider
>
> **4. Featured Courses Section**
>
> * Grid card layout (3–4 card mỗi row)
> * Mỗi course card gồm:
>
>   * Thumbnail
>   * Tên khóa học
>   * University / Instructor
>   * Rating ⭐
>   * Students enrolled
>   * Badge: Beginner / Intermediate
>
> **5. Browse Categories**
>
> * Card category dạng icon grid
> * Ví dụ:
>
>   * Data Science
>   * AI & Machine Learning
>   * Programming
>   * Business
>   * Marketing
>   * Design
>
> **6. Career Certificates Section**
>
> * Highlight các professional certificate
> * Card lớn với logo công ty (Google, IBM)
> * CTA: View Programs
>
> **7. Learning Path / Guided Projects**
>
> * Timeline hoặc step learning path
> * Beginner → Intermediate → Advanced
>
> **8. Testimonials**
>
> * User review card
> * Avatar + rating + comment
>
> **9. Call to Action Section**
>
> * Banner lớn
> * “Start learning today”
> * Button: Join for Free
>
> **10. Footer**
>
> * 4–5 column layout
> * Categories:
>
>   * Coursera
>   * Community
>   * More
>   * Mobile App
> * Social icons
> * Copyright
>
> **Design Style**
>
> * Modern EdTech UI
> * Rounded card
> * Soft shadow
> * Clean typography
> * Smooth hover animations
> * Material Design inspired
>
> **Output**
>
> * High fidelity UI
> * Desktop landing page
> * Clean grid layout
> * Professional SaaS style
